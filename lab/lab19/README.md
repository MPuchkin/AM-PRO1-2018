# Лабораторная работа №19
### Списки. Обработка списков

 1. Вставить значение в середину списка. Если количество значений нчетное, вставить после центрального элемента.
 2. Удалить заданное количество элементов, после первого элемента с заданным значением. 
 3. Вставить заданное значение пред каждым элементом с другим заданным значением.
 4. Заполнить список значениями из файла, расположив их в порядке не убывания значений.
