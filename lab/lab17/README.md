#Лабораторная работа №12
### Структуры в динамической памяти

Используя структуру `student`:
 1. Вставить новую запись в файл студентов не создавая нового файла.
 2. Удалить из двоичного файла данные о студентах с нулевым суммарным баллом, новый файл не создавать.
    __Указание:__ Необходимо загрузить данные из файла в память и затем перезаписать файл.
 3. Отсортировать данные в файле по суммарному баллу.
