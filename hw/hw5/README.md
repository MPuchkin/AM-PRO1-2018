# Домашняя работа №5

При решении задач необходимо использовать динамические массивы. Для каждой функции описать тесты.

Необходимо контролировать утечки памяти. После работы каждой функции необходимо вызывать `_CrtMemDumpAllObjectsSince` и проверять вывод в консоли VisualStudio.

1. Реализовать 3 функции сортировки массивов целых чисел различными алгоритмами. Протестировать их работоспособность с помощью `test_name_sort_int` из лабораторной работы №9.

2. Описать функцию для строящую таблицу зависимости времени сортировки от длины массива для набора функций согртировки
   `    N|     sort1|    sort1|    sort2| ...` для массивов длины `100 000 ... 10 000 000`.

3. Дан массив вещественных чисел `x` и фунция `f`. Реализовать функцию поиска элементов массива с минимальным расстоянием между точками `(x, y = f(x))`.

4. Дан массив вещественных чисел. Удалить из массива все элементы, меньшие среднего арифметического элементов массива.

5. Дан массив вещественных чисел. Вставить между каждой парой элементов значение разности этих элементов.
