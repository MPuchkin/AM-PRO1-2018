# Домашняя работа №9

Для каждой задачи написать тесты. Обработку ошибок реализовывать с помощью исключений.

1. Дан текстовый файл, содержащий матрицу целых чисел. Создать новый файл, содержащий транспонированную матрицу в том же формате. 
  Исходный файл имеет следующий формат:
  ```
    n m
    a11 a12 … a1m
    …
    an1 an2 … anm
  ```
2. Дан текстовый файл содержащий строки слов.
   - Посчитать количество вхождений заданного слова в файле.
   - Вывести самую длинную строку в файле.
   - Заменить все вхождения заданного слова в исходном файле, создав новый файл.
